Welcome to my submission for myEdOnline, recruitment test.

To test this, make sure you've node installed in your system. If not please follow this https://nodejs.org/en/download/ link.

After you've cloned or downloaded this project please direct to the downloaded folder through terminal or command prompt
and run the following command.

    node index.js

This command will work only if you're in the root folder.

After running this command you will see something like this in the terminal => Server running at: http://localhost:3000

Now navigate to, http://localhost:3000 this link in your browser to check the running application.
